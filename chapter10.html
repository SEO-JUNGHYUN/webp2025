Chapter 10 - HTTP 통신과 JSON

📡 HTTP 통신

  • HTTP: 브라우저와 서버 간의 요청/응답 통신 프로토콜
  • 주요 메서드:
    • GET - 데이터 조회
    • POST - 데이터 생성
    • PUT - 데이터 수정
    • DELETE - 데이터 삭제
  • Fetch API로 HTTP 요청 가능

🧾 JSON

  • JSON: JavaScript 객체와 유사한 구조의 데이터 포맷
  • 텍스트 기반으로, 서버-클라이언트 간 데이터 교환에 사용
  • JSON.stringify(): 객체 → JSON 문자열
  • JSON.parse(): JSON 문자열 → 객체

💻 예시 코드

  • fetch(url)로 HTTP 요청
  • response.json()으로 응답 데이터 파싱
fetch('https://example.com/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

📌 요약

  • HTTP는 웹 통신의 기반
  • JSON은 가장 많이 쓰이는 데이터 형식
  • fetch + JSON 조합은 JS의 핵심 통신 방식
