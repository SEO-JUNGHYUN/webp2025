Chapter 10 - HTTP 통신과 JSON

HTTP란...

  • 그냥 웹사이트랑 서버랑 주고받는 통신이라고 보면 됨
  • 예를 들어, GET은 정보 가져오기, POST는 보내기 이런 거임

JSON이 뭐냐면...

  • 자바스크립트 객체처럼 생긴 문자열 데이터
  • 서버랑 프론트가 서로 주고받기 딱 좋음
  • JSON.stringify()는 객체를 문자열로 바꾸고
  • JSON.parse()는 문자열을 다시 객체로 바꿈

예제 (그냥 fetch 써보기)

fetch('https://jsonplaceholder.typicode.com/posts')
  .then(res => res.json())
  .then(data => {
    console.log("데이터 왔음:", data);
  })
  .catch(err => {
    console.error("에러남", err);
  });

한 줄 요약

  • fetch()로 서버에 요청 보냄
  • .json()으로 응답 처리
  • 그걸 console.log로 확인 가능
