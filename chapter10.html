Chapter 10 - HTTP 통신과 JSON

HTTP란?

  • 브라우저와 서버가 통신할 때 사용하는 프로토콜
  • GET, POST, PUT, DELETE 같은 메서드로 요청을 보냄

JSON이란?

  • 자바스크립트 객체를 문자열 형태로 표현한 데이터 포맷
  • 서버와 데이터를 주고받을 때 자주 사용됨
  • JSON.stringify()로 객체를 JSON 문자열로 바꿈
  • JSON.parse()로 JSON 문자열을 객체로 변환

Fetch API 예시

fetch('https://jsonplaceholder.typicode.com/posts')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

정리

  • HTTP는 웹에서 데이터를 요청하고 응답받는 방식
  • JSON은 서버와 데이터를 주고받는 데에 많이 쓰이는 형식
  • 자바스크립트의 fetch()와 JSON 메서드를 조합해서 사용함
