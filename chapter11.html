Chapter 11 - Canvas

🖼️ Canvas란?

  • HTML5에서 제공하는 그래픽 영역 요소
  • 픽셀 기반의 그래픽을 자바스크립트로 직접 그릴 수 있음
  • 게임, 차트, 이미지 처리 등에 활용

📐 기본 구조

✏️ 주요 메서드

  • fillRect(x, y, width, height): 사각형 그리기
  • beginPath(), arc(), fill(): 원 그리기
  • moveTo(x, y), lineTo(x, y), stroke(): 선 그리기
  • clearRect(): 특정 영역 지우기

📌 요약

  • Canvas는 HTML에서 직접 그림을 그릴 수 있는 요소
  • getContext('2d')로 2D 도구를 받아 그림을 그림
  • 사각형, 원, 선 등을 조합해 복잡한 그래픽도 구현 가능
