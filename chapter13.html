Chapter 13 - API

API란?

  • Application Programming Interface의 약자
  • 프로그램이나 서비스 간에 데이터를 주고받기 위한 도구
  • 자바스크립트에서는 주로 fetch()를 통해 요청

웹 API

  • 브라우저가 제공하는 내장 API (예: document, location, fetch() 등)
  • 개발자가 자바스크립트로 웹 기능을 제어할 수 있게 해줌

오픈 API

  • 누구나 사용할 수 있도록 공개된 외부 서비스의 API
  • 예: 공공데이터 포털, 날씨 정보, 지도 API 등
  • 보통 URL과 키(API Key)를 이용해 사용

API 사용 예시

fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json())
  .then(data => {
    console.log(data);
  })
  .catch(err => console.error(err));

응답 데이터 사용하기

  • res.json()으로 응답을 JSON 형식으로 변환
  • 받은 데이터를 배열로 받아 반복문 등으로 출력 가능

예시: 사용자 목록 출력

fetch("https://jsonplaceholder.typicode.com/users")
  .then(res => res.json())
  .then(users => {
    users.forEach(user => {
      console.log(user.name + " - " + user.email);
    });
  });

정리

  • API는 데이터나 기능을 외부에서 사용할 수 있게 해주는 인터페이스
  • 웹 API는 브라우저 내장 기능, 오픈 API는 외부 서비스 연동용
  • fetch()와 .json()을 함께 사용해 쉽게 다룰 수 있음
