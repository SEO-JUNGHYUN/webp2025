Chapter 7

자바스크립트 DOM 활용 요약

  • document.createElement(요소명): 새로운 요소 노드 생성
  • document.createTextNode("텍스트"): 텍스트 노드 생성
  • appendChild(): 자식 노드를 부모 노드에 추가
  • 예제: 
    
orderButton.addEventListener("click", () => {
  let newP = document.createElement("p");
  let textNode = document.createTextNode(title.innerText);
  newP.appendChild(textNode);
  orderinfo.appendChild(newP);
});
  • createAttribute(): 속성 노드 생성 및 값 지정 
    
let srcNode = document.createAttribute("src");
srcNode.value = "images/wall.jpg";
요소노드.setAttributeNode(srcNode);
  • insertBefore(): 기존 자식 앞에 새 노드 삽입
  • remove(): 요소 자체 제거
  • removeChild(): 자식 노드 제거 
    
this.parentNode.parentNode.removeChild(this.parentNode);
