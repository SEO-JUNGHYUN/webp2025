Chapter 9

자바스크립트 객체 생성과 활용 심화 요약

  • 객체 만들기: 중괄호({})로 key-value 쌍 구성 
    
let student = {
  name: "Doremi",
  age: 17
};
  • 중첩 객체: 객체 안에 또 다른 객체나 함수 
    
let student = {
  name: "Doremi",
  score: {
    history: 85,
    science: 94,
    average: function() {
      return (this.history + this.science) / 2;
    }
  }
};
student.score.history;
student.score.average();
  • for...in문: 객체의 key를 순회 
    
let bag = {
  type: "backpack",
  color: "blue",
  size: 15
};
for (let key in bag) {
  console.log(`${key} : ${bag[key]}`);
}
  • 객체 메서드:
    • Object.keys(obj): 키 배열
    • Object.values(obj): 값 배열
    • Object.entries(obj): [키, 값] 배열
  • 프로토타입 상속: 객체 간 연결 
    
let parent = { canEat: true };
let child = {};
Object.setPrototypeOf(child, parent);
console.log(child.canEat); // true
